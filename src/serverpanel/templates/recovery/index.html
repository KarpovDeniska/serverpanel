Восстановление

Операции ниже переводят сервер в Rescue Mode и перезаписывают диски. Используй только при реальном отказе.

C-drive

Системный диск умер. Rescue → переустановка Windows из WIM → restore.ps1 из Storage Box.

D-drive

Диск данных умер. SSH к живой Windows → Initialize-Disk → restore_data.ps1.

Both drives

Оба диска. Сценарий C с флагом recover_d_drive = true.
